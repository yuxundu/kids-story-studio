# Kids Story Studio — 儿童个性化睡前故事生成器

让孩子的名字成为主角。输入姓名与喜好，2–5 分钟生成 10 页插画故事，可选中英双语，购买后获取高清 PDF 与音频（打印友好）。

- 在线演示（即将上线）：yourdomain.com
- 文档与模板：见 [starter/](starter/) 目录

## 快速开始（本仓库）
- 查看 [starter/README.md](starter/README.md) 获取项目定位与目录
- 按 [starter/docs/SPRINT_PLAN_7DAYS.md](starter/docs/SPRINT_PLAN_7DAYS.md) 从 D1 开始执行
- 需要最小可运行代码骨架（站点/HTML→PDF/Webhook/邮件）可在 Issue 提出，我将补全到 src/ 目录
