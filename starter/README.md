# Kids Story Studio — 儿童个性化睡前故事生成器

一句话：输入孩子姓名与喜好，自动生成带插画的个性化睡前故事（中英双语），购买后获取高清 PDF 与音频。

## 目标用户
- 3–8 岁孩子的家长
- 需要礼物/纪念的亲友
- 幼儿园/托育老师（集体故事时间）

## 核心卖点
- 个性化：孩子姓名、性格、宠物、城市等变量自动融入
- 稳定风格插画：一套固定风格，整本风格统一
- 双语：中文/英文版本可选
- 一键交付：下单即生成，PDF 下载 + TTS 音频

## 首发定价
- 单本：$9
- 主题套装（3 本）：$19
- 节日款：$12
